### ðŸ’¡ TypeScript Tips

1. **tsconfig.json**: Use strict mode
2. **Types vs Interfaces**: Prefer interfaces for objects, types for unions
3. **Avoid `any`**: Use `unknown` instead
4. **Type Guards**: Create custom type guards
5. **Generics**: Use for reusable components
6. **Utility Types**: Learn Pick, Omit, Partial, Required, Record
7. **Type Inference**: Let TypeScript infer when possible
8. **Zod**: Use for runtime validation + type inference
